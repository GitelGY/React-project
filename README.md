# Vistia Accessories 👜

פרויקט React המציג חנות אקססוריז עם מערכת ניהול משתמשים ומוצרים.

## תכונות עיקריות (Features)
* **מערכת התחברות (Authentication):** דף `Connect` המאפשר כניסה למערכת באמצעות שם משתמש וסיסמה.
* **ניהול מוצרים:** הצגת קטלוג (Products) ואפשרות להוספת מוצרים חדשים (AddProduct).
* **עגלת קניות:** ניהול רכישות (Cart) וממשק ביצוע קנייה (Buy).
* **ניהול מצב גלובלי:** שימוש ב-Context API ובאחסון מקומי (Storage) לשמירת נתוני המשתמש והעגלה.

## רכיבים מרכזיים
- `NavBar.jsx`: תפריט ניווט דינמי.
- `Connect.jsx`: טופס התחברות מאובטח.
- `AddProduct.jsx`: ממשק להרחבת מלאי החנות.
- `Context.js` & `Storage.js`: תשתית הנתונים של האפליקציה.

## הרצה
1. `npm install`
2. `npm start`
