# Vistia Accessories 👜

פרויקט React המציג חנות אקססוריז.

## מבנה הפרויקט
* **NavBar**: ניווט בין חלקי האתר.
* **Products & ProdList**: הצגת רשימת המוצרים מהחנות.
* **Cart**: ניהול עגלת הקניות של המשתמש.
* **Context API**: ניהול המידע של האפליקציה בצורה גלובלית.
* **About & Connect**: דפי מידע ויצירת קשר.

## הרצה מקומית
1. התקנת חבילות: `npm install`
2. הרצה: `npm start`
